== Description ==

Yala Blog is Simple Blog Theme , Install and Easy to Manage via Customizer, comes with Unlimited Color, Unlimited Google Fonts


== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Type "Yala Blog" in the search field and press "Enter" on your keyboard
3. Click the "Install" or "Activate" button to use your new theme right away.
